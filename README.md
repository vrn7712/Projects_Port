A rigorous catalog of interactive mechanics, digital engineering, and aesthetic exploration. This portfolio is built with a **Swiss Brutalist** design philosophy, prioritizing high-contrast typography, structural integrity, and fluid motion.

## ✦ The Architecture

The project index is structured into a three-tiered hierarchy to maintain a clear trajectory of work:

- **01 FEATURED:** The absolute best — high-impact, pinned projects showcased in massive bento grids.
- **02 CURRENTLY BREWING:** Active R&D. Projects currently in development, appearing as interactive cards.
- **03 PAST ARCHIVES:** A comprehensive, minimal list of all completed works, experiments, and hardware iterations.

## ✦ Aesthetic Identity

- **Typography:** High-contrast Serif for headers (Project titles) paired with technical Monospaced for descriptors.
- **Palette:** 
  - `Noir (#1A1A1A)` - Structural Base
  - `Ecru (#F2F0E9)` - Primary Background
  - `Vermillion (#EB5939)` - Action Highlight
- **Motion:** Cinematic introductory sequences, horizontal scroller archives, and spring-physics modal transitions.

## ✦ Tech Stack

- **Framework:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Vanilla CSS Flex/Grid for structural brute-force)
- **Animation:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Ticker, Custom Modifiers) & [Framer Motion](https://www.framer.com/motion/)
- **Tooling:** Antigravity, Claude Code, Codex, Gemini CLI, VS Code.
